# Selenium Scaffolder
Generate boilerplate code for automated testing using Selenium WebDriver.

## Installation

#### From Source
1. Clone the repo
```sh
git clone https://github.com/CaptnBigShot/Selenium.WebDriver.Extensions.git
```
2. CD to the project root directory and build
```sh
dotnet build
```
